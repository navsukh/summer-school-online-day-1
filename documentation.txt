HTML ELEMENTS

<form>: Creates a form for user input.
- <label>: Provides a readable label for each input field.
- <input>: Accepts various user inputs (text, email, tel, date, range, etc.).
- <textarea>: Allows multi-line text input (used for messages).
- <select> and <option>: Creates a dropdown menu for choosing one option.
- <br>: Adds line breaks for better layout without CSS.
- <h1> - <h6> : Used for section headings to improve readability.

<ul> : creates unordered list .
<li> : list elements.


type="text": Used for name because it's free-form user input.
type="email": Automatically validates proper email format.
type="tel" with pattern: Ensures only valid 10-digit phone numbers.
type="date": Lets users pick a date from a calendar UI, useful for appointments. 
type="range": Provides a slider for user ratings or satisfaction levels.


NAVIGATION STRUCTURE:

- The navigation bar is created using a <nav> element containing an unordered list <ul>.
- Each list item <li> contains an anchor <a> linking to different pages:
    - Home (index.html)
    - Portfolio (portfolio.html)
    - Contact (contact.html)
    - About (about.html)
- This structure is clean, responsive, and scalable for more links if needed.
- Users can easily navigate between pages without confusion.

This setup enhances user experience and allows for easy site expansion
